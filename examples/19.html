Custom Data Accessors, Part 2

If the data associated with a plottable needs to be accessed in a way different from the default method or the method defined by the chart.x() or chart.y() functions, the data accessors for the plottable can be set by using the functions of the same name on the plottable, e.g. line.x() or line.y(). Also note that areas may need to access more than one y-coordinate, and thus can have an additional y-accessor specified. See Bivariate Area Chart for an example.
