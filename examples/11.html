Changing Axis Locations, Part 2

The location of the x- and y- axes within the chart can be set using the .x_location() and .y_location() functions, respectively. If either is unspecified (or an invalid value is passed), the axis will be located in its default position, which is on the bottom for the x-axis and the left for the y-axis. In this example, the location of the x-axis is set to a percentage of the height of the chart and the location of the y-axis is set to a specific date on the x-axis. The following values can be used to set axis locations:

  • Locations - For the x-axis, 'top' and 'bottom' are valid. For the y-axis, 'left' and 'right' are valid.
  • Percentages - A string, e.g. '80%', indicating the percentage along the opposite axis's domain to place the axis.
  • Pixel Values - A string, e.g. '50px', indicating the number of pixels to translate the axis. Note that the axis will be translated away from the origin, as specified in margin conventions (i.e. the y-axis is translated from the top of the chart, not the bottom).
  • Specific Values - The value along the opposite axis at which to place the axis. Note that the value must be of the correct type, meaning that if the opposite axis scale uses date values, a date object must be passed (see this example).
