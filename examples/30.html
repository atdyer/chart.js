Chart Margins

Chart margins are the spacing between edges of the svg that contains the chart and the chart itself. The default margins, in pixels, are: 

var margin = {
    top: 20,
    right: 20,
    bottom: 20,
    left: 30
}

Margins can be specified using chart.margin(), which accepts an object with any of the top, right, bottom, left properties defined. Only those properties which are included will be updated. Larger margins can be particularly useful, as in this case, when very wide numbers are displayed with an axis.
