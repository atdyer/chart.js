Custom Data Accessors, Part 1

By default, all plotted data is assumed to be in array form 

            var data = [[0, 4], [1, 8], [2, 15], [3, 16], ...];

and plots will be generated by accessing the x- and y- values by index. However, if data is stored in some other form, for example 

            var data = [{x: 0, y: 4}, {x: 1, y: 8}, {x: 2, y: 15}, {x: 3, y: 16}, ...];

then we need to tell the chart how to properly access this data. The accessors can be set for the entire chart by calling the chart.x() and chart.y() functions. All plottables add to the chart will then use the newly defined accessors when referring to their data. The data accessors can also be set for individual plottables, see Custom Data Accessors, Part 2.
