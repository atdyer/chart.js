Adding Axis Labels, Part 1

Basic axis labels can be added using the .x_label() and .y_label() functions. The labels will move with the axes if their positions are changed using .x_location() or .y_location(), but they will not flip to the opposite side of the axis if a d3.axisTop() or d3.axisRight() is used. For more advanced control over axis labels, see Adding Axis Labels, Part 2.
