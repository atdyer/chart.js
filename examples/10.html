Changing Axis Locations, Part 1

By default, the x- and y-axes will be located at the bottom and left sides of the chart, respectively. If you use a d3.axisTop() or d3.axisRight() for either axis and want the axes to be oriented at the top and right side of the chart, you'll need to specify the location for each axis using .x_location('top') and .y_location('right') . There are additional values that can be passed to these functions to specify axis location, the full documentation of which can be found in the API Documentation, and an example of which can be found in Changing Axis Locations, Part 2.
