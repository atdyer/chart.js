The following methods are available for customizing legends:

  • legend.location() - Specify the legend position within the chart.
  • legend.box_width() - Specify the width of the color box associated with each legend item.
  • legend.box_height() - Specify the height of the color box associated with each legend item.
  • legend.box_padding() - Specify the vertical spacing between color boxes.
  • legend.margin() - Specify inner margins of the chart, see Legend Placement and Spacing.
  • legend.label_padding() - Specify the horizontal spacing between an item label and its color box.
