Linking Hover Events Between Charts

In order to connect hover events between multiple charts, a convenient linker function is provided. A linker is created by calling chart.linker(); note that an actual chart is not needed before creating the linker, simply call the function directly. The provided linker will honor the hover property of each plottable of every linked chart (i.e. the hover action will only occur if it has been enabled with .hover(true). Of course, if you don't want to use the provided linker, you can always link events between charts using chart or plottable events.
