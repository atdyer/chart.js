Sharing Axes Between Charts

This example is a recreation of the charts in Mike Bostock's Local Variables example.

The range of the values on the x- and y-axes can be controlled by using the .domain() and .range() functions, respectively. In order to share axes between multiple plots, calculate the extent of the data and set the domain/range as needed. If a domain or range is not specified, by default the axes will be fit to the data.

Additionally, custom components can be added to the chart after calling the chart() function. The chart() function returns the svg group that contains the drawn elements, and you are free to change them, delete them, or add new elements as you please. In this example, we add a label to each one.
